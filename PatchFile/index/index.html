임베디드 소프트웨어 최종 제작노트

Concept

DAW 혹은 MIDI Sequencer라고 불리는 소프트웨어를 구현하고 싶은 바람이 있었기에 이러한 프로젝트를 만들게 되었습니다. Daw, Midi Sequencer의 기능을 모두 구현하기지는 못했지만, 사용자가 쉽게 비트를 편집하고 저장할 수 있도록 하고, 최종적으로는 타임라인을 따라 재생되거나, 사용자가 저장한 비트를 실시간으로 트리거 할수 있는 패치를 제작하였습니다. 또한 visual patch를 키보드로 트리거하거나 arranger에 연결하여 조작할 수 있습니다.

Daw와 가상악기의 관계과 유사하게, 본체를 이루는 부분과 사운드를 만들어내는 부분을 분리하여 만들어내고자 하였으며, Ableton Live에서 Clip별로 트랙을 편집하고 트리거할 수 있는 기능을 참고로 하였습니다.

Quick Start

  1. DSP

Setup 섹션에서 DSP를 체크하여 켜져있도록 합니다.

2) 시작하기

Timer 섹션에서 Run을 클릭하면 beat가 시작되면서 소리를 들을수 있습니다.

3) 비트 불러오기/편집하기

Arranger 섹션의 preset을 클릭하거나, 각 트랙의 비트머신을 조작해서 저장된 비트를 불러오거나 비트를 편집할 수 있습니다.

4) 비주얼 패치 트리거

키보드 1부터 9, 그리고 '-' 버튼을 누르면 각 비주얼 패치를 끄거나 켤 수 있습니다.

5) 비주얼 패치 해제

키보드에서 '0'을 누르면 모든 비주얼이 해제됩니다.

6) Gem Window 열기/닫기

Setup섹션에서 WINDOW 를 트리거하면 창이 나타납니다.

또는 키보드에서 '/'을 누르면 gem창을 끄거나 켤수 있습니다.

7) Reset

Timer 섹션에서 Reset을 클릭하면 arranger의 시간이 0부터 다시 카운트됩니다.
